# This repo is for the Athelas Take Home Interview

# To Run :

Open two terminal tabs(one for FE, one for BE)

First run: ``` yarn ```

### FE: ``` yarn start ```
### BE: ``` cd api && yarn start-api ```

Visit http://localhost:3000 on your local browser and interact with the website!
